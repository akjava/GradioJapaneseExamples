Histgram Color Matching(scikit-image)

Body-to-body image color matching will never work; try face-to-face.keep eyes and hair color same
Usually default is best.To similar lighting angle,mirror(flip horizental) is important
Adding color paint(pink or green) can change hue(but not recommend to add target image)
modify tone-curve adjust brightness.However most of the case it's the lighting's issue, so there's not much we can do
